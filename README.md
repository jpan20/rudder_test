# Configuration
* Make sure motor voltage range: 11V ~ 13V
* Max current: 1A
* PWM mode: Locked Antiphase
* Encoder mode: Analog
* Control mode: Packet Serial Mode

# Run the test

* cd into test folder 
   ```
   $ python test.py
   ```
* rudder angle will be printed out
* type "quit" in command line will terminate the test
